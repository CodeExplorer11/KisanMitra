#  KisanMitra

KisanMitra is an AI-powered voice-first farming assistant designed for Indian farmers. The platform provides real-time weather updates, market prices, crop disease detection, soil analysis, government scheme guidance, and AI-based farming recommendations through voice or text interaction in Hindi and English.

Built using Retrieval-Augmented Generation (RAG) with Google Gemini, the system delivers personalized and context-aware agricultural assistance.

---

##  Features

-  Voice-based farmer assistance
-  Real-time weather insights & crop alerts
-  Market price analysis & selling recommendations
-  AI-powered crop disease detection
-  Soil health analysis using images/reports
-  Crop rotation & recovery suggestions
-  Government schemes & KVK/NABARD support
-  Women empowerment & rural support features

---

## Tech Stack

- **Frontend:** Streamlit
- **Backend:** Python
- **AI/LLM:** Google Gemini 1.5 Flash
- **Vision AI:** Gemini Vision
- **APIs:** OpenWeatherMap, OpenStreetMap
- **Voice:** Web Speech API
- **Deployment:** Streamlit Cloud
- **Version Control:** GitHub
